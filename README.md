# RemapKeyUI
Keyboard and mouse remapping tool

RemapKeyUI

键盘热键替换工具

功能列表：

1、映射成单次按键，比如按下A映射成B

2、映射成多次按键，比如按下A映射成多次B，直到A松开

3、映射成定时按键，比如按下A后，就会定时的自动多次按下B，直到再次按下A

4、映射成其他键按下，比如按下A后，映射成按下Shift+B，直到A松开时，Shift+B才松开；和功能2的区别在于这里只按下一次.
